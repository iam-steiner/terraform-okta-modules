# Terraform Modules for Okta Management

Terraform modules to create resources in Okta

## Folders structure

Each folder should contain a Terraform module with its own documentation

```shell
└── modules/
    ├── app_saml/
    │   ├── main.tf
    │   ├── ..
    │   └── README.md
    ├── app_oidc/
    │   ├── main.tf
    │   ├── ..
    │   └── README.md
    ├── group/
    │   ├── main.tf
    │   ├── ..
    │   └── README.md
    ├── policy_signon/
    │   ├── main.tf
    │   ├── ..
    │   └── README.md
    └── service_account/
        ├── main.tf
        ├── ..
        └── README.md
```
